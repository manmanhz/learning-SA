# 架构师考试：易错坑点汇总

## 1. 架构评估 (ATAM)
*   **权衡点 (Trade-off)**：涉及**两个**质量属性。一个变好，另一个变差。
*   **敏感点 (Sensitivity)**：涉及**一个**质量属性。一个设计决策对该属性影响极大。
*   **风险点 (Risk)**：架构设计中未确定的、可能导致失败的设计点。

## 2. 流水线计算 (Pipeline)
*   **不要全相乘**！
*   **公式**：$T = 第一条全长 (SUM) + (n-1) 	imes 周期 (MAX)$。
*   *注意*：如果选项里没有这个答案，再试 $(k+n-1) 	imes MAX$。

## 3. 子网划分 (Networking)
*   **永远记得减 2**：问“可用地址、可用主机、可用设备”时，一定要减去网络地址和广播地址。
*   **掩码转换**：
    *   /30 -> 4个IP (2可用)
    *   /29 -> 8个IP (6可用)
    *   /28 -> 16个IP (14可用)
    *   /27 -> 32个IP (30可用)

## 4. UML 关系
*   **空心菱形 (聚合)**：你可以走，我还在。(部门-员工)
*   **实心菱形 (组合)**：你走我也走，生死与共。(公司-部门、人-心脏)
*   **实线空心三角 (泛化)**：类继承类。
*   **虚线空心三角 (实现)**：类实现接口。
