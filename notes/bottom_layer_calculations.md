# 底层原理与高频公式 (战区三)

## 1. 流水线计算 (Pipeline)
*   **流水线周期 ($\Delta t$)**: MAX(各段耗时)
*   **总执行时间 $T$**: $(SUM_{各段耗时}) + (n - 1) * \Delta t$

## 2. 磁盘调度算法 (Disk Scheduling)
*   **FCFS**: 按顺序来。
*   **SSTF**: 每次找最近。
*   **SCAN**: 像电梯一样扫到底再回头。

## 3. 计算机网络计算 (子网划分)
*   **核心逻辑**: IP 地址总长 **32 位**。
*   **计算步骤**:
    1.  **主机位 $h$**: $32 - \text{斜杠后的数字}$。
    2.  **总 IP 数**: $2^h$。
    3.  **可用 IP 数**: $2^h - 2$ (去掉网络地址和广播地址)。
*   **常用速记**:
    *   /24 = 256个IP (254可用)
    *   /25 = 128个IP (126可用)
    *   /26 = 64个IP (62可用)
    *   /27 = 32个IP (30可用)
    *   /28 = 16个IP (14可用)
    *   /29 = 8个IP (6可用)
    *   /30 = 4个IP (2可用)

## 4. 系统可靠性计算
*   **串联系统**: $R = R_1 \times R_2 \times ... \times R_n$
*   **并联系统**: $R = 1 - (1-R_1) \times (1-R_2) \times ... \times (1-R_n)$
