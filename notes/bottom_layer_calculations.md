# 底层原理与高频公式 (战区三)

## 1. 流水线计算 (Pipeline)
*   **流水线周期 ($\Delta t$)**：取所有段中耗时最长的那一个。
*   **总执行时间 $T$** (假设 $n$ 条指令, $k$ 个执行阶段):
    *   **理论公式**: $(t_1 + t_2 + ... + t_k) + (n - 1) 	imes \Delta t$
        *   *口诀：第一条指令全速跑，剩下的指令每隔一个周期出一个结果。*
    *   **实践公式**: $(k + n - 1) 	imes \Delta t$
*   **吞吐率 $TP$**: $n / T$
*   **最大吞吐率 $TP_{max}$**: $1 / \Delta t$

## 2. 磁盘调度算法
*   **先来先服务 (FCFS)**：按顺序找。
*   **最短寻道时间优先 (SSTF)**：谁离磁头近先找谁 (容易导致两端的请求“饥饿”)。
*   **扫描算法 (SCAN/电梯算法)**：磁头来回扫，不回头。

## 3. 存储层次与 Cache
*   **Cache 命中率 $h$**：$t_a = h 	imes t_1 + (1 - h) 	imes t_2$
    *   $t_a$：平均访问时间。
    *   $t_1$：Cache 访问时间。
    *   $t_2$：主存访问时间。
