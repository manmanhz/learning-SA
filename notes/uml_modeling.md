# UML 建模与辨析 (战区二)

## 1. 静态模型：类图关系 (Class Diagram)
*   **泛化 (Generalization)**：实线 + 空心三角。**类继承类** (Is-a)。
*   **实现 (Realization)**：虚线 + 空心三角。**类实现接口**。
*   **聚合 (Aggregation)**：空心菱形。弱整体。
*   **组合 (Composition)**：实心菱形。强整体。

## 2. 动态模型：用例图关系 (Use Case)
*   **包含 (Include)**：<<include>>。必选动作。
    *   *方向*：主用例 -> 被包含用例。
*   **扩展 (Extend)**：<<extend>>。可选动作。
    *   *方向*：**扩展用例 -> 主用例** (反向箭头)。
*   **泛化 (Generalization)**：用例之间也可以继承 (如：支付用例 -> 微信支付用例)。

## 3. 各种行为图辨析 (必考)
*   **顺序图/序列图**：强调**时间顺序**。
*   **状态图**：描述**单个对象**的状态机。
*   **活动图**：强调业务流程，支持**并行**。
*   **通信图**：强调对象间的结构关系 (旧教材叫协作图)。
